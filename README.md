# k6-performance-tests

Repositório para testes de performance com k6.

## Estrutura

- `tests/`: Scripts de teste organizados por módulos.
- `.github/workflows/`: Workflows do GitHub Actions.

## Como executar

1. Instale o k6: https://k6.io/docs/get-started/installation/
2. Execute um teste: `k6 run tests/01-modulo-fundamentos/hello-world.js`
